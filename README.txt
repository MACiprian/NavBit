Instalarea librariilor

Selenium

1. Cream un fisier ".py" principal.
2. Adaugam extensiile IntelliSense (Pylance), Linting, Debugging (multi-threaded, remote), Jupyter Notebooks, code formatting, refactoring, unit tests
3. Selectam versiunea corecta de Python Interpreter (3.10.5)
4. Deschidem un nou terminal
5. Rulam comanda "pip install selenium" pentru a adauga librariile
6. Verificam instalarea cu "pip show selenium" pentru a fi siguri de versiunea corecta

Librariile aditionale

Numpy

1. Rulam in terminal "pip install numpy"

OpenCV

1. Rulam in terminal "pip install opencv-contrib-python"

pyautogui

1. Rulam in terminal "pip install pyautogui"

pyaudio

1. Rulam in terminal "pip install pipwin"
2. "pipwin install pyaudio"
3. "pip install PyAudio"
4. Daca iesirea arata "Requirement already satisfied" instalarea a avut succes
